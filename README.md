# coin-platform

Monorepo for:

- `apps/web-agent`: virtual coin market + prediction UI (lightweight-charts).
- `packages/ui`: reusable React component library for the app.
- `packages/tokens`: design tokens.
- `packages/utils`: shared utilities.

## Quick start

```bash
pnpm install
pnpm dev
```

## Useful scripts

```bash
pnpm build
pnpm test
pnpm storybook
```
